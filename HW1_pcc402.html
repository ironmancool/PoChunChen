Answer: a,b

Which of the following are true?
a. If P(A ∪ B) = P(A), then B ⊂ A
b. If P(A ∩ B) = P(B), then B ⊂ A

\[\begin{equation}\begin{split}P(A \cup B) & = P(A)+P(B)-P(A \cap B)\end{split}\end{equation}\] \[\because in \,option\,a\, P(A ∪ B) = P(A) \] \[\therefore P(B) = P(A \cap B)\\B \subset A\] \[\because in \,option\,b\, P(A ∩ B) = P(B) \] \[\therefore P(A) = P(A \cup B)\\B \subset A\]

2.16

\[Using\, the\, definitions\, of\, r!\, and\, n!\, (Equation 2.23)\, and\, \binom{n}{r}\] \[ prove \,\binom{n}{r} = \binom{n-1}{r-1}+\binom{n-1}{r}\]

\[\binom{n}{r} = \frac{n!}{r!(n-r)!}\]

\[\begin{equation}\begin{split}\frac{n!}{r!(n-r)!} &= \frac{(n-1)!}{(n-1-(r-1)!(r-1)!} + \frac{(n-1)!}{(n-1-r)!r!} \\&= \frac{(n-1)!}{(n-r)!(r-1)!} + \frac{(n-1)!}{(n-1-r)!r!}\\&= \frac{(n-1)!}{(n-r)!r!}*r + \frac{(n-1)!}{(n-r)!r!}*(n-r)\end{split}\end{equation}\] Both side divided by r!(n-r)!

\[ \begin{equation}\begin{split}n! &= (n-1)!*r+(n-1)!*n-r\\&=(n-1)!*(n-r+r)\\&=(n-1)!*n\\&=n!\end{split}\end{equation}\]

Assume that leaf biomass (in grams) from the plant Salix arctica follows the pdf,
f(x) = 2(x + 1)^-3 for x > 0.
Find the probability of a leaf being between 0 and 3 g, that is, P(0 < X < 3).
Solve by hand using calculus. Verify your result using R.

\[\int_0 ^3 2(x+1)^{-3} dx = \\ 2*\int_0 ^3 (x+1)^{-3} dx\] Substitute u=x+1

\[2*\int_0 ^3 u^{-3} dx = \\ -\int_0 ^3 u^{-2} dx =\\ - (x+1)^{-2}|3,0 +C = 15/16 = 0.9375\]

integrand <- function(x) {2*(x+1)^-3}
integrate(integrand, lower = 0, upper = 3)
